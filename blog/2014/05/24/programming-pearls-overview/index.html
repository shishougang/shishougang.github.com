Programming Pearls Overview

May 24th, 2014 | Comments

全书分为3部分：

  • 预备知识：总的概括代码涉及的知识点，比如算法和数据结构的重要性和合理 性，如何写出正确的代码并证明，如何测试代码，性能评估代码，debug等。
  • 性能：先是介绍一些估算的技巧，比如72法则，利特尔法则(Little’s law) 等，之后展开代码算法的性能，如何调试代码使得性能更好或更省空间。
  • 产品：讲解具体的算法，如排序，搜索等。

书本和习题大部分代码实现。
